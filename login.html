Configurer Google OAuth

Pour activer "Continuer avec Google", tu dois créer une application OAuth 2.0 sur Google Cloud Console et coller tes identifiants ici.

1
Va sur console.cloud.google.com/apis/credentials → clique Créer des identifiants → ID client OAuth 2.0 → Type : Application Web.
2
Dans URI de redirection autorisés, ajoute exactement :
http://localhost:3000/api/auth/google/callback
3
Copie l'ID client et le Secret client générés et colle-les ci-dessous.
